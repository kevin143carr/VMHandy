# VMHandy Storyboard And Shot Planning

## 30-Second Shot List

1. Finder window full of `.pvm` and `.vmwarevm` bundles, messy and slow.
2. VMHandy opens.
3. User switches between `Parallels` and `VMware Fusion`.
4. Select remote folder and local folder.
5. Click `Copy To Local`.
6. Registered VM appears.
7. Click `Launch VM`.
8. VM boots.
9. Click `Copy VM To Remote` or `Delete Local Copy`.
10. End card: `VMHandy for macOS`

## 60-Second Shot List

1. Messy Finder windows with `.pvm` and `.vmwarevm` bundles.
2. User rubbing forehead.
3. VMHandy opens.
4. Provider switcher: Parallels to VMware Fusion.
5. Remote folder selected.
6. Local folder selected.
7. `Copy To Local`.
8. `Register`.
9. `Launch VM`.
10. VM desktop appears.
11. `Copy VM To Remote`.
12. Clean final UI shot.
13. End card.
